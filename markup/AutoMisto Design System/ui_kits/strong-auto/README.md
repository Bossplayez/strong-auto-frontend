# Strong Auto — UI Kit

Click-thru recreation of strong-auto.ua. Open `index.html`.

## Screens
- Home — hero, category filter chips, hot offers grid, calculator + auto-in-transit + news strip
- Search — left filter sidebar + 3-up listing grid
- Listing (in Ukraine) — gallery + spec table + seller card
- Listing (in transit) — gallery + spec table + price + calculator
- Auction listing — gallery + bid panel + characteristics

## Components
- `LeftRail.jsx` — vertical icon nav
- `TopHeader.jsx` — logo + nav + auth
- `ListingCard.jsx`, `PriceTag.jsx`, `StatusTag.jsx`
- `Calculator.jsx`, `FilterSidebar.jsx`
- `Footer.jsx`, `Screens.jsx`
